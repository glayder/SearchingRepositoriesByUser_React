# Searching Repositories by User
Searching repositories and descriptions by user from github

### Instalação da aplicação
```bash
# Clone este repositório
$ git clone <https://github.com/glayder/Searching-Repositories-by-User.git>

# Acesse a pasta do projeto no terminal/cmd
$ cd Searching-Repositories-by-User/

# Instale as dependências
$ yarn

# Execute a aplicação
$ yarn start

# O servidor inciará na porta:3333 - acesse <http://localhost:3000>
```

### Teste da aplicação
```
# Execute o camando para executar os testes
yarn test;
```
